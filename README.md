# 넵무새 - 성공적인 사회초년생활 카톡 코치

`넵무새`는 대학원생/사회초년생이 교수님, 선배, 상사, 외부 파트너에게 카톡 답장을 보낼 때
실수 없는 비즈니스 커뮤니케이션을 돕는 MCP 서버입니다.

## 핵심 문제

- "네 알겠습니다"로 끝내기엔 부족한 답장 품질
- 어려운 용어(IP, 과제 용어, 업무 약어) 이해 부족
- 읽씹/재촉/지연 사과에서 관계 손상 위험
- 합의사항을 명확히 남기지 않아 생기는 책임 이슈

## MVP 핵심 기능

1. 상대별 톤 프로파일링
2. 전송 전 리스크 검사
3. 톤별 답장 생성
4. 어려운 용어 해설
5. 정중한 역질문 생성
6. 책임소재 확인 문안 생성
7. 지연 사과 만회 문안 생성

## 프로젝트 구조

- `nepmoosae_server.py`: MCP 엔트리포인트
- `risk_engine.py`: 리스크 분석 로직
- `tone_profiles.py`: 상대별 톤 정책
- `templates.py`: 상황별 템플릿
- `term_helper.py`: 용어 추출/해설
- `tool_descriptions.py`: 툴 설명 텍스트
- `docs/FEATURE_SPEC.md`: 기능 명세
- `docs/KAKAOTALK_TOOL_TESTS.md`: 카톡 스타일 테스트 프롬프트
