# PlayMCP 카톡 사용 예시 (니돈내산 v1.1)

PlayMCP / 카카오톡에서 그대로 복사해 테스트할 수 있는 프롬프트 10개입니다.

---

## 1. 온보딩

```
협찬 처음인데 뭐부터 해?
```

에이전트가 적성 테스트 안내 → 맞춤 팁 → 인기 협찬 순으로 연결합니다.

---

## 2. 적성 테스트 + 프로필 저장

```
블로그 맛집 서울 살아. 적성 테스트하고 프로필 저장해줘
```

적성 유형(`food_explorer` 등)과 `region: 서울`이 저장되면 이후 검색에 자동 반영됩니다.

---

## 3. 오늘의 인기 협찬

```
오늘 인기 협찬 5개 보여줘
```

신청자 수 기준 인기순 TOP 5 + 지역·D-day·경쟁률 테이블.

---

## 4. 지역 + 업종 검색

```
서울쪽 레스토랑 협찬 추천해줘
```

`search_campaigns_by_need`가 지역(서울)·업종(맛집)을 파싱해 맞춤 캠페인을 반환합니다.

---

## 5. 지역 숙박

```
부평 근처 숙박 체험단 있어?
```

제목 `[인천 부평]` 태그·주소 기반으로 부평 숙박 캠페인을 찾습니다.

---

## 6. 마감 임박 협찬

```
마감 하루 안 남은 협찬 5개 알려줘
```

`get_urgent_campaigns` — D-0/D-1 마감 임박순 정렬.

---

## 7. 니즈 탐색 (배송형)

```
강아지 간식 배송형 협찬 찾아줘
```

키워드 + 카테고리 매칭, 없으면 인기순 fallback.

---

## 8. 체험가치 · 시장가 비교

```
revu-1367756 시장가랑 비교해줘
```

네이버 쇼핑 API로 시장 최저/최고/평균가와 제공 혜택을 비교합니다.

---

## 9. 신청 한마디

```
이 캠페인 신청 한마디 써줘 blog.naver.com/dablro12
```

캠페인 + 블로그 채널 분석 기반 3문장 신청 코멘트 생성.

---

## 10. 팁 + 캠페인 체이닝

```
선정률 올리는 팁 알려주고, 서울 맛집 방문형 협찬 3개랑 상위 1개 신청 링크까지
```

`get_sponsorship_tips` → `search_campaigns_by_need` → `get_campaign_link` 순서로 연결.

---

## E2E 체이닝 시나리오 (심화)

```
1. 적성 테스트 (블로그, 맛집, 서울, 주1회 가능, 방문형, 사진, 경험없음)
2. 프로필 저장
3. 맞춤 팁 auto
4. 서울 맛집 방문형 캠페인 3개
5. 마감 임박 3개도 같이 보여줘
6. 상위 1개 신청 문구 + 링크
```
