**서비스명:** 니돈내산 - 협찬 받고, 추천까지 한 번에  
**MCP 식별자:** `nidonnaesan`  
**버전:** v1.2  
**대상:** 블로거 · 인스타 리뷰어 · 협찬 초보자 · 체험단 입문자

---

## 1. 서비스 한 줄 요약

니돈내산은 **카카오톡에서 말하듯이 협찬을 물어보면, 맞는 협찬을 골라 표로 추천해 주는 도우미**입니다.

여러 체험단 사이트를 직접 돌아다니지 않아도, **인기 협찬 · 조건 맞춤 · 마감 임박 · 경쟁 적은 협찬**을 한곳에서 확인할 수 있습니다.

> 본 서비스는 **협찬 정보 추천**과 **신청 문구 초안**을 제공합니다.  
> 실제 신청, 리뷰 작성, 광고 표기는 사용자 본인이 진행해야 합니다.

## 3. 핵심 기능 (5가지)

### ① 협찬 추천 (메인 기능)

사용자가 원하는 조건에 맞는 협찬을 골라 **표 형태**로 보여줍니다.

- **초보 추천** — 경쟁이 비교적 여유 있고 체험가치 괜찮은 협찬
- **니즈 맞춤** — "서울 맛집", "부평 숙박", "뷰티 배송형" 등 자연어 검색
- **마감 임박** — 오늘·내일 마감(D-0/D-1) 협찬
- **경쟁률 낮은 순** — 신청자 대비 모집 인원이 여유 있는 협찬 우선

**4컬럼 슬림 표 (기본)**

| 협찬 제품명 | 경쟁률 | 제품가격 | 바로가기 |
|------------|--------|----------|----------|
| … | 3/10 (0.3:1) | 50,000원 | [신청](링크) |

### ② 인기·마감·검색 (세부 조회)

추천 외에도 목적별로 따로 조회할 수 있습니다.

- **오늘의 인기 협찬** — 신청자가 많은 인기순 TOP N
- **니즈 탐색** — 지역·업종·키워드 파싱 후 맞춤 검색
- **마감 임박** — D-day 기준 급한 협찬만

### ③ 가치 판단

협찬이 "받을 만한지" 참고할 수 있는 정보를 제공합니다.

- **체험가치 라벨** — 제공 혜택 대비 높음/보통/낮음
- **경쟁률** — 신청자/모집인원, 여유/보통/치열
- **시장가 비교** — 네이버 쇼핑 기준 최저·최고·평균가 (API 연동 시)

### ④ 협찬 노하우 팁

운영자가 검증한 체험단 팁 **29개**를 주제별로 안내합니다.

- 선정률 올리는 법, 플랫폼 비교, 블로그 지수
- 광고 표기, 포스팅 누락, 리뷰 작성 가이드
- 네이버 SEO, AI 브리핑 인용 기준 등

초보·중급에 맞춰 **다음 읽을 팁**도 자동 추천합니다.

### ⑤ 프로필·적성 (맞춤 필터)

한 번 저장해 두면 이후 추천·검색에 **기본 조건**으로 반영됩니다.

| 저장 항목 | 반영 내용 |
|----------|----------|
| 지역 (`region`) | 서울, 부평 등 지역 필터 |
| 관심 업종 (`preferred_category`) | 맛집, 뷰티, 숙박 등 |
| 매체 (`preferred_media`) | 블로그, 인스타릴스 등 |
| 방문/배송 (`preferred_type`) | 방문형, 배송형 |
| 블로그 URL (`channel_url`) | 신청 문구 초안 작성 시 사용 |

> 적성 테스트만 하면 저장되지 않습니다. **`set_reviewer_profile`로 저장**해야 반영됩니다.

## 6. 사용 시나리오

### 시나리오 A — 협찬 완전 초보

```
1. "협찬 처음인데 뭐부터 해?"
   → 적성 테스트 안내

2. 적성 테스트 후 프로필 저장
   → 지역·관심 업종 저장

3. "경쟁률 낮은 협찬 5개 추천해줘"
   → easy_pick + 4컬럼 표

4. "체험단 선정률 올리는 방법 알려줘"
   → 맞춤 팁 제공
```

### 시나리오 B — 지역·업종 맞춤 찾기

```
"서울쪽 레스토랑 협찬 추천해줘"
→ 지역(서울) + 업종(맛집) 파싱 후 표 출력

"경쟁 덜한 거 위주로"
→ sort_by=low_competition 적용
```

### 시나리오 C — 마감 임박

```
"마감 하루 안 남은 협찬 알려줘"
→ D-0/D-1 협찬, 마감 임박순 정렬
```

### 시나리오 D — 가치 판단 후 신청

```
1. "이 협찬 시장가랑 비교해줘"
2. 관심 있으면 "신청 링크 알려줘"
3. (선택) "신청 한마디 써줘"
```

## 8. 데이터 출처

| 구분 | 출처 | 용도 |
|------|------|------|
| 협찬 캠페인 | 체험단모음 API | 목록·상세·신청 링크 |
| 시장가 | 네이버 쇼핑 Search API | 가격 비교 (선택 연동) |
| 채널 | 네이버 블로그 RSS | 채널 분석·신청 문구 |
| 팁 | blogreviewzip.com | 선정·플랫폼·SEO 등 29개 주제 |
| 프로필 | 서버 SQLite | 지역·업종 등 필터 저장 |

캠페인 데이터는 **Docker 빌드 시 동기화**되며, 실시간 정확도는 원본 API에 따릅니다.

## 9. 제한 사항 및 유의사항

- **신청 대행 없음** — 링크와 정보만 제공, 직접 신청해야 합니다.
- **리뷰·광고 표기** — [공정위 심사지침](https://www.ftc.go.kr) 준수는 사용자 책임입니다.
- **신청 문구** — AI 초안이며, 본인 채널에 맞게 수정 후 제출하세요.
- **시장가** — 네이버 API 미설정 시 비교 기능이 제한될 수 있습니다.
- **프로필** — OAuth 사용자 구분 없이 단일 키로 저장될 수 있어, 다중 사용자 환경에서는 주의가 필요합니다.
- **캠페인 페이지** — 일부 사이트는 상세 페이지 자동 수집이 어려워, 제품명·쇼핑 검색으로 맥락을 보완합니다.
